Getting Started with Kotlin

A Kotlin Project in IntelliJ IDEA

In IntelliJ IDEA, configure a new project with the following settings:

  • Name - Pick a suitable name for your project
  • Location - The folder you use for your coding
  • Language - Kotlin
  • Build System - IntelliJ
  • JDK - Any recent version, e.g. 18 or higher

Hello World!

This is a classic 'Hello World' program. Typically we write these to check that everything it setup and working correctly.

Replace any example code in your porject with this code:


                fun main() {
                    println("Hello, World!")
                }

Press the green 'play' arrow next to the main function to run the code. You should see `Hello World` appear in the output panel...


                Hello, World!

Note that the main() function ('fun' is short for function) is always the entry point for any Kotlin program.
