Displaying Text and Data to the Console

The Console

When you run a console program, output messages are shown as text in an area called the console. The console will show:

  • System messages (e.g. exit codes, 0 means 'success')
  • Error messages (i.e. when your program crashes)
  • Program output (text you intend to be seen)

For example, a successful program...


$ java -classpath kotlin-stdlib.jar MainKt

Hello World!

$ Process finished with exit code 0

Or a program that throws an error...


$ java -classpath kotlin-stdlib.jar MainKt
! Exception in thread "main" NumberFormatException
! at MainKt.main(Main.kt:17)

$ Process finished with exit code 1

Output of Lines of Text

Outputting text to the console, line-by-line is done via the println function:

println(text)
Outputs the given text and moves the cursor down to the next line ⏎

So, this code...


                println("Hello World")    // Goes down to next line
                println("from Kotlin!")   // Goes down to next line
                println()                 // Outputs a blank line
                println("How lovely!")    // Goes down to next line

... would give this output


                Hello, World ⏎
                from Kotlin! ⏎
                ⏎
                How lovely! ⏎
                |

Output of Text on Same Line

Outputting text to the console so that it is all on the same line is done via the print() function:

print(text)
Outputs the given text, leaving the cursor at the end of the text

So, this code...


                print("This text ")   // Stays on same line
                print("is all ")      // Stays on same line
                print("on the ")      // Stays on same line
                print("same line")    // Stays on same line

... would give this output


                This text is all on the same line|

Output of Variables

Variable values can be easily added into other text using String Templates. Variables are simply placed into the string preceded by a $ symbol:


                val forename = "Jimmy"
                val surname = "Smith"

                println("Hello, $forename $surname!")

Would give this output:


                Hello, Jimmy Smith!

Function calls, etc. can also be placed inside a string, but must be wrapped in {...} brackets:


                println("Welcome, ${forename.get(0)}. $surname")

Would give this output:


                Welcome, J. Smith
