Kotlin

A modern programming language for the JVM

Kotlin is a cross-platform, statically typed, general-purpose high-level programming language with type inference.

Kotlin provides a modern alternative to Java, interoperating fully with Java and the Java Virtual Machine (JVM). It is the recommended choice for Android app development. Kotlin code is more concise and readable than the equivalent Java code, and is a pleasure to program in.

An Example Kotlin Program


                fun main() {
                    while (true) {
                        print("Name: ")
                        val name = readln()
                        // Quit if no name
                        if (name.isBlank()) break

                        print("Mark: ")
                        val mark = readln().toIntOrNull()

                        print("Feedback: ")
                        val feedback = readln()

                        // Work out the grade
                        val grade = when (mark) {
                            in 0..24   -> "Not Achieved"
                            in 25..49  -> "Achieved"
                            in 50..74  -> "Merit"
                            in 75..100 -> "Excellence"
                            else -> "unknown"
                        }

                        println()
                        println(name)
                        println("-".repeat(name.length))
                        println(grade)

                        // Show feedback as a list if given
                        if (feedback.isNotBlank()) {
                            feedback.split(", ").forEach {
                                println(" - $it")
                            }
                        }
                        println()
                    }
                }
