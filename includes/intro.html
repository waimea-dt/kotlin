Kotlin is a cross-platform, statically typed, general-purpose high-level programming language with type inference.

Kotlin provides a modern alternative to Java, interoperating fully with Java and the Java Virtual Machine (JVM). It is the recommended choice for Android app development. Kotlin code is more concise and readable than the equivalent Java code, and is a pleasure to program in.
